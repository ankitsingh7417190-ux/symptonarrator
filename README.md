
# 🩺 SymptoNarrator: AI-Powered Health Symptom Narrator

SymptoNarrator is an AI-driven healthcare platform designed to streamline patient symptom reporting using NLP entity extraction and speech-to-text processing.

---

## 🚀 Key Features

* **High-Accuracy NLP Pipeline:** Powered by `spaCy` and `TensorFlow` to extract medical entities and symptoms with **95% accuracy**.
* **Speech-to-Text Input:** Integrated voice-input features that reduce manual data entry by **60%**.
* **Secure Backend API:** Robust RESTful architecture capable of handling patient records with optimized response times.
* **Scalable Data Management:** Utilizes MongoDB/PostgreSQL for flexible schema management and high-performance querying.

---

## 🛠 Tech Stack

* **Frontend:** React.js, JavaScript, HTML/CSS
* **Backend:** Python, Flask, REST APIs
* **AI & NLP:** TensorFlow, spaCy, Speech-to-Text API
* **Database:** PostgreSQL / MongoDB
* **Tools:** Git, Postman

## ⚙️ Installation & Setup

### 1. Clone the Repository

git clone [https://github.com/YOUR_USERNAME/symptonarrator.git](https://github.com/YOUR_USERNAME/symptonarrator.git)
cd symptonarrator

### 2. Backend Setup
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
3. Frontend Setup
Bash
cd ../frontend
npm install
npm start
